Bottom Tab Bar Demo

Scroll down to see the fixed bottom navigation bar

Features

  • Hamburger Menu: Click the bars icon to open a slide-up menu with additional options
  • Main Navigation: 4 primary tabs + central FAB for quick access
  • Fixed Bottom Position: Always visible at the bottom of the screen
  • Floating Action Button (FAB): Central button with gradient and hover effects
  • Active State: Highlights the current page
  • Smooth Animations: Slide-up menu with overlay
  • Responsive Design: Adapts to different screen sizes
  • Safe Area Support: Works with iOS notches and Android navigation bars

Customization

You can easily customize:

  • Colors: Change gradient colors in the CSS
  • Icons: Replace Font Awesome icons with your own
  • Links: Update href attributes to match your pages
  • FAB Action: Customize the showAddModal() function

Implementation

To add this to your pages:

  1. Add <link rel="stylesheet" href="bottom-tab-bar.css"> to your <head>
  2. Add class="has-tab-bar" to your <body> tag
  3. Copy the HTML from bottom-tab-bar-snippet.html before your closing </body> tag
  4. Customize the links and functions to match your app

Tips

  • The FAB (center button) rotates on hover for a playful effect
  • Active tabs have a subtle pulse animation
  • Each tab can have its own color when active
  • The bar has rounded top corners for a modern look
